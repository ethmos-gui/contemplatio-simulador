# Simulador Contemplatio

Simulador público de pacotes editoriais e distribuição profissional.

**URL pública**: https://ethmos-gui.github.io/contemplatio-simulador/

## Estrutura

- `index.html` — landing page (HTML + CSS inline)
- `simulador.calc.js` — módulo de cálculo (preços, deltas, mensagens WhatsApp)
- `contemplatio-logo.png` — logo da marca

## Atualizar preços

Edite os blocos `PACKAGE_BASE`, `EXTRAS` e `DIST` em `simulador.calc.js` e faça push.
GitHub Pages republica automaticamente em ~1 min.

## Fonte

Mantido junto ao projeto editorial em [contemplatio/materiais/](https://github.com/ethmos-gui/contemplatio).
